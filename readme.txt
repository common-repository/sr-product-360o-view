== Description ==

SR Product 360&deg; View Provides you with the capability to add 360&deg; image rotation to your store Products. Now present your Product to your customers with extra detail and in a more engaging way. No more boring images slideshow when you can give a complete 360&deg; view of your Products.

**New Features:**
1. Custom Product Image Size
2. Custom Popup Size
3. Show 360 icon over Product image instead of thumbnails
4. Shortcode support
5. Customize all icons width and height
6. Select your own icons
7. Product individual settings
8. Image Scaling on popup
9. 360&deg; views for variable Products(variations)
10. Bulk import 360&deg; images
11. And more...

== Installation ==

1. Install & activate WooCommerce plugin, if not activated already.
2. Upload the entire `sr-product-360o-view` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
